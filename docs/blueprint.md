# **App Name**: Campus Cafe Connect

## Core Features:

- User Authentication: Secure user login and registration with email/password and alternative options.
- Menu Display: Categorized menu display (South Indian, North Indian, Refreshments, Chats).
- Item Selection and Quantity: Ability to add items to the cart with specified quantities.
- Real-time Inventory Check: Check if the items are available using an external API.
- Cart Management: Display added items with quantities and calculate the total price.
- Order history: History of orders made at the cafe in previous times.

## Style Guidelines:

- Primary color: Warm orange (#FF7043) to evoke feelings of warmth, appetite, and a welcoming atmosphere, reflecting the cafe's environment.
- Background color: Light beige (#F5F5DC) for a clean, inviting backdrop that complements the orange.
- Accent color: Muted yellow (#FFB300), for key interactive elements and highlights to draw user attention.
- Body and headline font: 'PT Sans', a humanist sans-serif for a balance of modern appeal and warmth, ideal for both headlines and body text. 
- Simple, outlined icons for category navigation and item representation.
- Clean, grid-based layout for easy navigation and clear display of menu items and cart information.
- Subtle animations for item additions to the cart and updates to the total price.
